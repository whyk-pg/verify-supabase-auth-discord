# Verify Supabase Auth Discord

## 本リポジトリの目的
Auth.jsに移行するまでの間の暫定措置として、Supabase Authを利用してDiscord認証を行なう

## 本リポジトリの達成目標
- [ ] Node.jsでSupabase Authが動くか確認する
  - [ ] 特定サーバー以外はエラーを返せるか確認する
  - [ ] 特定ユーザー以外はエラーを返せるか確認する
- [ ] DenoでSupabase Authが動くか確認する
  - [ ] 特定サーバー以外はエラーを返せるか確認する
  - [ ] 特定ユーザー以外はエラーを返せるか確認する
